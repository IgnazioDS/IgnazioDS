# Ignazio De Santis

![Ignazio De Santis banner](./assets/profile-banner.svg)

I build AI systems where retrieval, orchestration, evaluation, and automation actually have to work under real operating conditions.

Most of my work sits at the boundary between applied AI and product infrastructure: code-aware assistants, agent execution systems, eval harnesses, developer tools, and data pipelines with observability built in from the start.
